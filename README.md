# Spoti-like mobile

A Spotify clone app made during an interview process.

See my web version on [Spotify-clone](https://github.com/bastiendmt/spotify-clone)

## Installation

Get an access token

```sh
deno run --allow-read --allow-net auth.js
```

`npm install`

`expo start`
